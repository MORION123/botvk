# 🤖 VK Mutual PR Bot

Бот для взаимного пиара в ВКонтакте. Помогает набирать репосты, лайки, подписчиков и комментарии через систему взаимных заданий.

## ✨ Функционал

- ✅ Создание заданий (репосты, лайки, подписки, комментарии)
- ✅ Выполнение чужих заданий
- ✅ Система баллов
- ✅ Личный кабинет с балансом
- ✅ VIP статус (удвоенные баллы)
- ✅ Вывод баллов
- ✅ Топ пользователей
- ✅ Статистика
